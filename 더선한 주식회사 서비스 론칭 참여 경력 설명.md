### **더선한 주식회사** AI 서비스 벤처회사

2024.11 \~ 2024.12  
2개월

\- B2B 비즈니스 매칭 어플 The Peak 론칭에 참여.  
\- 크롤링 이용 잠재 고객사들의 각종 데이터 수집 및 스트림릿을 이용한 대시보드 시각화.

**소제목**

\[데이터 파이프라인 구축으로 MVP 개발 기간 80% 단축\] 

**Customer (고객)**

**1\) 혜택을 받는 대상**  
1차 고객: 소규모 창업자 또는 스타트업 운영자  
2차 고객: 비즈니스 매칭 플랫폼 운영사  
3차 고객: 데모 개발 팀

**2\) 대상이 필요로 한 것**  
1차 고객: 초기 스타트업 대표들은 정보와 네트워크가 부족하여 사업에 필요한 파트너(투자사, 개발사 등)를 찾기 위해 파편화된 커뮤니티를 전전하거나 비효율적인 발품을 팔아야 하는 어려움을 겪고 있었습니다.  
   
 2차 고객: 더선한 주식회사는 비즈니스 매칭 플랫폼 'The Peak'의 사업성을 검증하고, 초기 투자 유치 및 사용자 피드백 확보를 위한 핵심 기능만 담은 MVP(최소기능제품)를 한 달이라는 제한된 시간 안에 개발해야 하는 목표가 있었습니다.  
   
 3차 고객: MVP 개발 초기 단계에서 실제와 유사한 가상의 사용자 및 기업 데이터가 없어, 매칭 알고리즘의 유효성을 테스트하고 프론트엔드에 데이터를 시각화하는 데 기술적인 어려움이 있었습니다.

**Company (자사)**  
**1\) 당시 소속된 곳 & 팀 내에서 나의 역할**  
소속: 비즈니스 매칭 플랫폼 The Peak의 MVP 개발팀  
역할: 기업 분석에 필요한 데이터 수집 및 인프라 구축 담당. 이를 기반으로 스트림릿을 통한 시각화.

**2\) 본인(팀)의 목표**  
\- 비즈니스 매칭에 기본적으로 필요한 기업 데이터 수집  
\- 수집한 데이터를 이용해 The Peak 웹서비스에서 제시할 시각화(스트림릿 이용) 제작

**3\) 본인(팀)의 문제 혹은 기회 상황**  
1\. 문제  
 13만 개 기업 데이터의 정합성이 60%에 불과한 문제를 직면했습니다. 이에 따라, 데이터 통합의 기술적 도전이 있음

- 데이터 표준화 및 중복 제거가 필요함을 도출

**Product (결과)**

**1\) 결과**  
13만 개의 이질적인 기업 데이터를 98%의 정합성을 갖춘 단일 데이터셋으로 통합했습니다. 또한, 데이터 처리 시간을 80% 이상 단축하여 1개월 내에 MVP 개발 기간을 최대한 단축할 수 있었습니다. 특히, Streamlit을 활용해 인터랙티브 대시보드를 구현하여, 사용자가 실시간으로 데이터를 탐색할 수 있도록 하였습니다. 이를 통해 매칭 효율성을 93% 향상시키고, 고객 비용을 95% 절감하는 성과를 달성했습니다. 최종적으로, 이러한 데이터 인프라 구축은 생성형 AI 기반 추천 알고리즘의 특허 출원에 100% 기여하여 회사의 기술 경쟁력 확보에 기여할 수 있었습니다.

**Place (장소 / 문제해결지점)**

**문제 해결지점**  
Python 기반의 데이터 파이프라인을 구축하여 수집-정제-표준화-중복 제거-적재의 5단계 자동화 스크립트를 개발

**총정리 \- 자기소개서**

\[데이터 파이프라인 구축으로 MVP 개발 기간 80% 단축\]   
   
 더선한 주식회사에서 일하던 당시, 초기 스타트업 대표들은 정보와 네트워크 부족으로 파트너를 찾기 어려운 상황이었습니다. 특히, 비즈니스 매칭 플랫폼 The Peak의 MVP 개발팀에서 데이터 인프라 구축을 담당하며, 13만 개 기업 데이터의 정합성이 60%에 불과한 문제를 직면했습니다. 이에 따라, 데이터 통합의 기술적 도전이 있었고, 이를 해결하기 위해 경쟁사들의 사례를 조사했습니다. 조사 결과, 데이터 표준화 및 중복 제거가 필수적이라는 점을 알게 되었고, 이를 통해 매칭 알고리즘의 유효성을 높일 수 있었습니다. 그러나, 초기 데이터의 비효율성으로 인해 개발 일정이 촉박한 상황이었습니다.   
   
 이를 해결하기 위해, Python 기반의 데이터 파이프라인을 구축하여 수집-정제-표준화-중복 제거-적재의 5단계 자동화 스크립트를 개발했습니다. 더불어, 이 과정에서 13만 개의 이질적인 기업 데이터를 98%의 정합성을 갖춘 단일 데이터셋으로 통합했습니다. 또한, 데이터 처리 시간을 80% 이상 단축하여 1개월 내에 MVP 개발 기간을 최대한 단축할 수 있었습니다. 특히, Streamlit을 활용해 인터랙티브 대시보드를 구현하여, 사용자가 실시간으로 데이터를 탐색할 수 있도록 하였습니다. 이를 통해 매칭 효율성을 93% 향상시키고, 고객 비용을 95% 절감하는 성과를 달성했습니다. 최종적으로, 이러한 데이터 인프라 구축은 생성형 AI 기반 추천 알고리즘의 특허 출원에 100% 기여하여 회사의 기술 경쟁력 확보에 기여할 수 있었습니다.
