## Project Overview

이 레포지토리는 B2B 비즈니스 매칭 플랫폼 **The Peak**의 MVP 개발을 위해
기업 데이터를 크롤링하고, 정제·통합하여 Streamlit 대시보드로 시각화한 프로젝트입니다.

## RAG & HyDE Concept Note

본 프로젝트 이후, 비즈니스 매칭 고도화를 위해 RAG + HyDE 기반 추천/챗봇 구조를
이론적으로 설계한 글을 작성했습니다. (The Peak 2단계 로드맵 개념 설계에 해당)

- 제목: HyDE로 RAG 향상시키기: 이론과 적용 방안
- 역할: 공동 저자 (이론 정리, 비즈니스 적용 시나리오 기획)
- 링크: https://www.koreaodm.com/인공지능/hyde로-rag-향상시키기-이론과-적용-방안/

이 글에서는
- RAG 구조와 한계,
- HyDE(Hypothetical Document Embeddings)의 개념,
- The Peak와 같은 B2B 매칭 서비스에 RAG+HyDE를 어떻게 적용할지
를 정리하고, 본 레포의 데이터 인프라 위에서 어떤 기능을 확장할 수 있는지 제안합니다.
